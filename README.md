# Coffee Recommender System ☕

## Descripción
Aplicación web desarrollada como proyecto académico que recomienda tipos de café
utilizando un sistema basado en reglas con **Prolog**, integrado a una aplicación web moderna.

La aplicación interactúa mediante un **chatbox**, donde el usuario responde preguntas
y recibe una recomendación generada por el motor lógico en Prolog.

---

## Tecnologías utilizadas

### Frontend
- React
- Vite
- Tailwind CSS
- React Router DOM

### Backend
- Node.js
- Express.js
- API REST personalizada

### Lógica
- Prolog
- Integración Prolog–JavaScript mediante Engine.js

---

## Funcionalidades
- Interfaz web interactiva
- Chatbox para interacción con el usuario
- Comunicación frontend–backend
- Motor de recomendaciones basado en reglas
- Respuesta dinámica desde Prolog
